# Phaser 2D Starter Template

A minimal playable Phaser 3 starter project.

## Files Generated

- `index.html` - Entry point
- `src/main.js` - Game bootstrap
- `src/scenes/BootScene.js` - Asset loading scene
- `src/scenes/GameScene.js` - Main gameplay scene
- `src/entities/Player.js` - Player placeholder
- `package.json` - Project metadata and dependencies
- `README.md` - Project documentation

## Running the Project

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Open browser to http://localhost:8080
```
