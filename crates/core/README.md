## What You Can Modify

- **Player speed** - Edit `src/entities/Player.js`, change `this.speed = 200`
- **World size** - Edit `src/scenes/GameScene.js`, modify `setBounds(0, 0, 1600, 1200)`
- **Platform positions** - Edit the `platformPositions` array in GameScene.js
- **Number of coins** - Change `repeat: 11` in `createCollectibles()`

## Adding Real Assets

1. Create an `assets/` folder in your project root
2. Add your images there
3. Load them in `BootScene.js` instead of generating textures:

```javascript
// Replace this:
graphics.generateTexture('player', 32, 32);

// With this:
this.load.image('player', 'assets/player.png');
```

## Troubleshooting

**Game doesn't load?**
- Check browser console for errors (F12)
- Ensure all files are in the correct directories
- Verify `npm install` completed without errors

**Player doesn't move?**
- Click on the game area first to focus it
- Check that arrow keys aren't being intercepted by your OS

**Want to start over?**
- Delete the project folder and regenerate with PeridotCode
